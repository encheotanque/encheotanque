Status de Arquitetura & Microserviços

Análise Técnica - "Enche o Tanque" | Atualizado em Abril de 2026

1. Comparativo de Stack (Planejado vs. Executado)

Abaixo apresentamos a evolução e pivotagem da tecnologia durante a esteira de desenvolvimento do projeto, otimizada para lidar dinamicamente com coordenadas topológicas e interfaces Mobile-First de alta performance.

Stack Planejada (Inicial)

  • Frontend: React genérico ou HTML/JS.
  • Backend API: Motor Padrão genérico.
  • Banco de Dados: MySQL (Oracle Cloud).
  • Integração: API Google Maps crua.
  • Processador (Worker): Web Scraper Python genérico.

Stack Atual (Executada)

  • Frontend: React 18 + Vite + TailwindCSS + Animacões Framer Motion.
  • Backend API: Node.js (Express Middleware) + Cálculo Haversine nativo.
  • Banco de Dados: Oracle Cloud MySQL.
  • Integração MX: Google Maps + Interpolação Matemática RGB SVG. Deep link rotas (/maps/dir) nativo.
  • Processador (Worker): PHP rodando via Crontab (Servidor Nacional).
